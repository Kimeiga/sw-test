SW Test

This is a small project I made with the following goals:

  1. Have a subdomain that when you visit it, it automatically redirects you to a different place on the domain
  2. Have a service worker on that subdomain that caches a JavaScript file that is needed for the page to be redirected to
  3. See if the service worker can cache that JavaScript file upon going to the redirect page so that it can be served from cache on the page to be redirected to
